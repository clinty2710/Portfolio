Capstone Project: Hotel Website

This was a project I started in my last term of my software development degree from August to October of 2023. We were a team of two - I was the team leader, designer and front end developer for our page and my partner did all the back end development.

The goal was to create a website for a Hotel that was in the San Juan Islands in the Pacific Northwest. The site needed to include a reservation system and a way for a user to create a profile, Look up or edit their reservations and gain knowledge about the hotel's surrounding area.

We Prototyped, designed, and built the app for the website. We also created a simple way to make a reservation which was similar to selecting a seat at a movie theater. The user was able to view the layout of the hotel and select a desired room to make reservations. Of course that would entail showing rooms that are already taken/booked.

We worked on this website for 2 1/2 months before we turned it in for a final grade.

Technologies:
  • - Python
  • - Flask
  • - HTML & CSS
  • - Java/Javascript
  • - Bootstrap
  • - MySQL Databases
  • - Image Maps/ImageMapster
